# Forendors brand system — always apply to generated images

## Color palette (light mode — default for all slide graphics)

- Background (page): #F2F2F2 — warm neutral light gray
- Light (cards, panels): #FAF5EC — WARM CREAMY off-white, NOT cool gray-white
- Main Red (primary accent, CTA, focal element): #D2452D
- Main Black (primary text / dark elements): #1F1F1F
- Mid Black (secondary text): #303030
- Icon gray (icons, muted lines): #6D6D6D
- Border (thin separators): #D8D8D8
- Positive (success): #4F8E09
- Negative (error/warning): #E51139
- Yellow (highlight background): #FFF3DA
- Orange (attention badge): #F6AD12

## STRICT color rules
- NEVER use cool gray-white (#F0F0F6), navy, dark blue, lime green, neon anything.
- Main Red #D2452D is the ONLY primary accent per image (use sparingly, one focal element).
- Secondary accent colors (Positive green, Orange, Negative) may appear ONLY when the concept demands it (e.g., success state, warning). Not decorative.
- The dominant tone of every image is warm light neutrals: #F2F2F2 background with #FAF5EC cards. Warm beige-cream, never cold blue-gray.

## Typography
- Nunito Sans (Google Fonts), weights 300 / 400 / 700 / 800.
- When text IS needed: Nunito Sans Bold 700, tight tracking, #1F1F1F color.
- Default: NO TEXT in generated images. No labels. No captions. No HEX codes. No placeholder words. No numbers. No measurements. No watermarks. No signatures.

## Visual language
- Flat 2D design. No drop shadows. No gradients. No glows. No 3D.
- Rounded corners: 10–20px for cards/boxes, fully pill-shaped (999px radius) for button-like shapes.
- Borders: 1px thin, #D8D8D8. Not heavy strokes.
- Generous whitespace. Editorial magazine aesthetic. European/Czech design-studio feel.
- Minimal elements per image. Clean composition. Nothing busy.

## Iconography
- Flat vector icons, outlined or simple filled shapes.
- Icon fill/stroke color: #1F1F1F charcoal or #6D6D6D gray.
- Accent color on one icon only (Main Red) when it's the focal/danger element.
- Consistent stroke weight across all icons in one image.

## What NOT to do (hard NOs)
- NO dark navy / black backgrounds.
- NO lime green, neon green, electric cyan.
- NO cool gray-white backgrounds (#F0F0F6 family). Always warm cream #FAF5EC or gray #F2F2F2.
- NO Silicon Valley cyberpunk styling.
- NO 3D renders, NO glossy surfaces, NO photographic realism.
- NO stock-photo clipart, NO generic tech artwork.
- NO rainbow gradients, NO busy patterns, NO drop shadows.
- NO text/labels/HEX codes/numbers unless explicitly requested.
